## [7.6.0]

### Added

- Opening category for NACamDoorTag
- Schedule modification
- Bticino MyHome Server 1 scopes
- NLPD - Drivia dry contact

- support for Legrand garage door opener (NLJ)

## [7.5.0]

### Added

## [7.3.0]

### Added

### Changed

- Minor code clean ups

### Fixed

- Response handling issues
